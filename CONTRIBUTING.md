# Contributing

Contributions are welcome and will be fully credited.

## Pull Requests

1. **Fork and clone:** Fork the repository and clone it to your local machine.
2. **Create a branch:** Create a new branch for your feature or bug fix.
3. **Write tests:** Ensure your changes are covered by tests.
4. **Static analysis:** Run `composer phpstan` to ensure type safety.
5. **Submit PR:** Push your branch and open a pull request on GitHub.

## Bug Reports

Please open an issue on GitHub for any bug reports or feature requests.
